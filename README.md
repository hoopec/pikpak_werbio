### 如何运行

1. 下载`werbio.py`文件到本地用 Python 运行。
2. 运行后提示什么错误就安装什么库，例如: `pip install requests`。
3. 运行成功后复制网址到浏览器打开，即可运行在网页执行邀请程序，可搭建部署在服务器运行。
